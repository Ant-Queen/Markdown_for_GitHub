# 이모지 (emojis)

## 이모지 사용 (Using emojis)

> for GitHub  

`:EMOJICODE:`, 콜론, 이모지 이름 순으로 입력하여 글에 이모지를 추가할 수 있습니다.

`@octocat :+1: This PR looks great - it's ready to merge! :shipit:`

![Screenshot of rendered GitHub Markdown showing how emoji codes for +1 and shipit render visually as emoji.](https://docs.github.com/assets/cb-17228/mw-1440/images/help/writing/emoji-rendered.webp)

<kbd>:</kbd>을 입력하면 제안된 이모지 목록이 표시됩니다. 입력할 때 목록이 필터링되므로 원하는 이모지를 찾았으면 **Tab** 또는 **Enter** 키를 눌러 강조 표시된 결과를 완성합니다.

사용 가능한 이모지 및 코드의 전체 목록을 보려면 Emoji-Cheat-Sheet[Emoji-Cheat-Sheet](https://github.com/ikatyang/emoji-cheat-sheet/blob/master/README.md)를 확인하세요.

## ✨ 유니코드 이모지 (Unicode emojis)

유니코드 이모지는 이모지 코드가 아닌 유니코드 문자를 사용하여 표시됩니다. 예를 들어, `:smile:` 대신 `😊`를 사용할 수 있습니다.  
유니코드 이모지는 대부분의 최신 운영 체제 및 플랫폼에서 지원됩니다. 유니코드 이모지를 입력하려면 다음 방법 중 하나를 사용할 수 있습니다:
- **Windows**: `Win + .` (윈도우 키와 마침표 키)를 눌러 이모지 선택기를 엽니다.
- **Mac**: `Control + Command + Space`를 눌러 이모지 선택기를 엽니다.
- **모바일 장치**: 키보드에서 이모지 아이콘을 탭하여 이모지 선택기를 엽니다.
유니코드 이모지는 GitHub, Slack, Twitter 등 다양한 플랫폼에서 지원되며, 텍스트에 시각적 요소를 추가하는 데 유용합니다.
예:
- 웃는 얼굴:😊
- 엄지 척:👍
- 빨간 하트:❤️
- 파티 팝퍼:🎉
- 불꽃놀이:🎆
- 별똥별:🌠
- 피자 조각:🍕
- 축구공:⚽
- 비행기:✈️
- 음악 노트:🎵
- 카메라:📷
- 책:📚
- 커피 컵:☕
- 태양:☀️
- 달:🌙

이모지를 사용할 때는 접근성 및 문화적 차이를 고려하는 것이 중요합니다. 모든 사용자가 이모지를 동일하게 해석하지 않을 수 있으므로, 의사소통에 혼란을 피하기 위해 적절한 맥락에서 사용하는 것이 좋습니다.
추가 리소스:

- [Get Emoji](https://getemoji.com/)

- [Unicode 공식 사이트 - Full Emoji List, v17.0](https://unicode.org/emoji/charts/full-emoji-list.html)

- [Wikipedia - List of emojis](https://en.wikipedia.org/wiki/List_of_emojis)



📚 전체 목록 확인 방법
- Unicode 공식 사이트: Full Emoji List (v17.0)에서 모든 유니코드 이모지를 볼 수 있습니다. 이 페이지는 각 이모지의 코드 포인트, 이름, 다양한 플랫폼별 렌더링 이미지를 제공합니다.
- Emojipedia: Emojipedia에서도 유니코드 버전별 이모지 목록을 쉽게 검색하고 복사할 수 있습니다.
- Wikipedia: List of emojis 문서에서는 유니코드 블록별로 이모지 개수와 분류를 설명합니다. 예를 들어, Unicode 17.0 기준으로 총 3,953개의 이모지가 정의되어 있습니다.


https://en.wikipedia.org/wiki/List_of_emojis



| 구분 | GitHub 이모지 코드 (`:emoji_code:`) | 유니코드 이모지 (⚙️) |
|------|--------------------------------------|-----------------------|
| **입력 방식** | `:+1:`, `:shipit:`, `:smile:` 처럼 콜론으로 감싸 입력 | 그냥 복사/붙여넣기 또는 키보드에서 직접 입력 |
| **렌더링 환경** | GitHub, Slack, Discord 등 **지원하는 플랫폼에서만** 변환됨 | 모든 최신 OS/브라우저에서 기본적으로 표시됨 |
| **표현 범위** | GitHub 전용 커스텀 이모지 포함 (예: `:shipit:` → 다람쥐) | 표준 유니코드 이모지만 사용 가능 |
| **자동완성 지원** | `:` 입력 시 제안 목록이 뜨고 Tab/Enter로 선택 가능 | 자동완성 없음, 직접 입력해야 함 |
| **호환성** | GitHub 밖에서는 코드 그대로 보일 수 있음 (`:+1:`) | 어디서나 동일하게 보임 (⚙️) |

👉 정리하면:
- GitHub 이모지 코드는 플랫폼 특화 기능이라 GitHub 같은 곳에서만 변환되고, 커스텀 이모지를 쓸 수 있다는 장점이 있어요.
- 유니코드 이모지는 어디서나 동일하게 보이는 범용성이 강점입니다.
