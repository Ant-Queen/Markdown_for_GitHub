# 텍스트 인용 (Quoting text)

<kbd>></kbd>를 사용하여 텍스트를 인용할 수 있습니다.

```markdown
Text that is not a quote

> Text that is a quote
```

Text that is not a quote

> Text that is a quote

<kbd>></kbd> 붙은 텍스트는 왼쪽에 세로선으로 들여쓰기되고 회색 형식으로 표시됩니다.

```markdown
Text that is not a quote

> first-level
>> second-level
>>> third-level
>
> first-level
```

Text that is not a quote

> first-level
>> second-level
>>> third-level
>
> first-level
